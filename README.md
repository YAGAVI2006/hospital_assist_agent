# рҹ“– Overview

**Starlight Medical Center вҖ“ AI Hospital Information Assistant** is a modern full-stack healthcare web application that combines a premium hospital website with an intelligent AI receptionist.

The system enables patients to quickly access hospital information, explore departments, find doctors, learn about medical services, and receive instant AI-powered assistance through natural language conversations.

The AI assistant uses **Google Gemini** together with the hospital's structured data to provide accurate, contextual, and user-friendly responses.
