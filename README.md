## 🤖 About Starlight AI

**Starlight Medical Center – AI Hospital Information Assistant** is a full-stack healthcare web application that combines a modern hospital website with an intelligent AI-powered virtual receptionist.

Built using **React**, **Node.js**, **Express**, and **Google Gemini AI**, the system helps patients quickly access hospital information, explore departments, discover doctors, understand medical services, and receive instant AI-powered responses through natural language conversations.

The AI assistant uses the hospital's structured knowledge base together with Gemini AI to deliver accurate, contextual, and user-friendly healthcare assistance.

Traditional hospital websites provide static information that often requires patients to search through multiple pages.

This project demonstrates how **Generative AI** can improve digital healthcare experiences through intelligent, interactive, and user-friendly assistance.

# 🏗️ System Architecture

The **Starlight Medical Center AI Assistant** follows a modular full-stack architecture that separates the frontend, backend, AI service, and hospital knowledge base.

The architecture is designed to be scalable, maintainable, and easy to extend with additional hospital services.

---

## 🏛️ High-Level Architecture

```text
                           🌐 USER
                              │
                              ▼
                ┌────────────────────────┐
                │   React + Vite Client  │

                └─────────────┬──────────┘
                              │
                     HTTP REST API
                              │
                              ▼
                ┌────────────────────────┐
                │   Express.js Backend   │
```
